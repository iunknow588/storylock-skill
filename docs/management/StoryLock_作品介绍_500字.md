易安与 StoryLock 是一套面向 Agent 场景的本地授权与安全执行方案。核心目标是把签名授权、密码填充和敏感对象访问从远端服务拉回用户可控的本地设备，同时保留 Agent 协作的便利性。

系统采用三层技术架构：第一层处理文件加密与操作授权；第二层负责底层操作的封装与远程访问风险评估；第三层作为远程网关，统一针对第三方 Agent 的访问入口。

关键机制是：远端 Agent 可以提出请求，私人智能助理封装授权，StoryLock 本地核心完成授权，用户始终在本地设备上掌握最终确认权，避开了复杂的密钥记忆及风险操作。

技术实现上，第一层处理故事草稿、润色与题集强度评估；第二层负责对象强度判断、九宫格验证、短时授权和审计日志；第三层仅暴露 requestSignature 和 requestPasswordFill 两个接口，对结果递归脱敏后返回。当前支持 Android 与 Windows 本地宿主，已完成端到端自测验证。

一句话概括：远端 Agent 可以提出请求，私人智能助理封装授权，StoryLock 本地核心完成授权，用户始终在本地设备上掌握最终确认权，避开了复杂的密钥记忆及风险操作。
