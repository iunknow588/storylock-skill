# StoryLock 流程图目录

这个目录提供一套从下载到最终应用的图文材料，内容按“总览 -> 下载 -> 设置 -> 空模式与授权 -> 保存与应用”组织。

## 文件清单

1. [01_总览图.png](./01_总览图.png)  
   StoryLock 完整操作总览图，展示获取包、进入空模式、选择包目录或 `vault.stlk`、挑战解锁、加载当前包内容、保存当前包并应用的流程。
2. [02_下载流程图.png](./02_下载流程图.png)  
   StoryLock 下载流程图，展示包如何下载、复制到本地并进入后续设置阶段。
3. [03_设置流程图.png](./03_设置流程图.png)  
   StoryLock 设置流程图，展示统一入口、路径归一、冲突检查、空模式与待解锁状态。
4. [04_使用与授权流程图.png](./04_使用与授权流程图.png)  
   StoryLock 使用与授权流程图，展示空模式进入、挑战解锁、当前包内容加载、学习、保存当前包和最终应用。
5. [StoryLock_完整操作流程说明.md](./StoryLock_完整操作流程说明.md)  
   对应的文字版流程说明。
6. [StoryLock_从下载到应用操作手册.md](./StoryLock_从下载到应用操作手册.md)  
   从下载、设置、解锁到保存应用的编号操作说明，已引用完整 26 张图片。
7. [StoryLock_界面截图说明.md](./StoryLock_界面截图说明.md)  
   对应的界面截图与说明图索引。
8. [ui-screenshots](./ui-screenshots)  
   Host 与 StoryLock 关键界面截图目录，并包含包结构、路径连续性、保存应用边界和视频分镜细节图。当前共 26 张图片。
9. [StoryLock_流程图使用说明.md](./StoryLock_流程图使用说明.md)  
   图中内容范围说明。

## 推荐阅读顺序

1. 先看 [01_总览图.png](./01_总览图.png)
2. 再看 [02_下载流程图.png](./02_下载流程图.png)
3. 对照 [ui-screenshots/08_storylock_package_structure.png](./ui-screenshots/08_storylock_package_structure.png) 检查包目录
4. 对照 [ui-screenshots/11_download_package_landing.png](./ui-screenshots/11_download_package_landing.png) 到 [ui-screenshots/26_save_as_new_package_boundary.png](./ui-screenshots/26_save_as_new_package_boundary.png) 准备视频分镜素材
5. 然后看 [03_设置流程图.png](./03_设置流程图.png)
6. 最后看 [04_使用与授权流程图.png](./04_使用与授权流程图.png)
7. 再看 [StoryLock_从下载到应用操作手册.md](./StoryLock_从下载到应用操作手册.md)
8. 最后对照 [StoryLock_界面截图说明.md](./StoryLock_界面截图说明.md)
