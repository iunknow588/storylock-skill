# 易安对外分众介绍后续备忘

日期：2026-06-18

## 原评审口径

易安展示 StoryLock 的端侧可信执行边界：敏感对象留在 Android，本网站只负责入口、编排、状态和脱敏返回。

## 原合作方口径

易安提供可集成的下载、绑定、注册和远程请求接口，适合接入需要用户本地授权的签名或密码填充场景。

## 原开发者口径

开发者可以通过 `GET /api/storylock-gateway` 获取连接配置，通过 `POST /api/storylock-gateway` 提交标准 envelope，并由 Android 宿主完成本地执行。

## 后续处理建议

1. 如需评审材料，放在 `docs/ref` 或演示讲解文档中。
2. 如需合作方材料，单独做接入说明，不放在普通用户首页。
3. 如需开发者材料，单独做 API 文档，不放在普通用户首页。
4. 线上首页优先回答普通用户关心的问题：如何下载、如何绑定、如何确认、如何判断是否在线、遇到问题怎么办。
