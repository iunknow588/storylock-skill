# StoryLock 术语表

## 目的

统一设计文档中的核心术语，避免同一概念反复换说法。

| 术语 | 定义 |
| --- | --- |
| Skill | 具有明确输入输出契约的能力单元 |
| 主 Skill | 直接对应用户任务的 Skill |
| 内部 Skill | 作为底层构件存在的 Skill |
| 编排示例 | 说明如何串联多个 Skill 的示例对象 |
| Host | 提供本地敏感执行边界的运行时 |
| Gateway | 远程 Agent 访问本地能力的受控接口 |
| Challenge | 用于授权验证的问题或挑战流程 |
| Session | 挑战通过后建立的短时访问状态 |
| Secret Object | 被 StoryLock 保护的秘密对象 |
| 故事处理 Skill | 负责编辑、修改、完善故事内容的本地 Skill |
| 故事访问 Skill | 负责授权读取 / 写回故事对象的本地 Skill |
| 远程 Skill | 面向第三方请求、包装、编排、委托的远程能力 |
| 代理签名 | 第三方请求本地完成签名，而非自己持有私钥 |
