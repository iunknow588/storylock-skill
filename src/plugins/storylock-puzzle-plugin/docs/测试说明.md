# 测试说明

插件测试只覆盖九宫格展示和输入收集职责。

## `builds_cells_from_draft`

验证 `create_open_challenge_from_draft`：

- 按 `required_cells` 生成题目
- 保留 `cell_id` 和 `prompt_text`
- 只读取选项 `text`
- 将选项补齐到 9 个
- 不依赖正确答案字段

## `toggles_selection_by_normalized_value`

验证 `toggle_selection`：

- 点击非空选项后加入用户选择
- 再次点击同一选项后取消选择
- 使用归一化值判断是否为同一选项

## `normalizes_answer_text`

验证 `normalize_answer` 会去除首尾空格并转为小写。

## 不再测试的内容

本插件不再测试答案验证，因为验证不属于九宫格展示 UI：

- 不测试 `authorize_open`
- 不测试 `expected_answers`
- 不测试 `isCorrect`
- 不测试解锁授权

这些应在外部验证服务或宿主适配层中单独测试。
