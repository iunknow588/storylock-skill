  1. 下载易安 App 安装包。
  2. 在 Android 设备上安装并允许来自当前来源的安装。
  3. 从易安打开绑定链接，按本地设备提示完成绑定。
  4. 完成首次注册后，回到站点查看 App 在线状态。
