安全边界

云端入口 只负责展示、下载、中转请求和返回最小状态。
私人智能助理 解释来源、内容和风险，不直接读取故事存储。
StoryLock Core 离线处理敏感内容，授权前不展示答案和受保护对象。

遇到问题先判断

  1. 下载后先核对文件名、大小和校验值。
  2. 选择包时确认只指向一个包根目录或一个 vault.stlk。
  3. 空模式看不到内容是正常保护，挑战通过后才加载。
  4. 保存后 Host 应继续使用同一路径，另存为新包需要手动切换。
